### install
#### shell 
```
curl -o-  https://raw.githubusercontent.com/lemonadezZ/FourLeafClover/master/bin/install.sh | bash
```
#### centos
```
yum install -y   docker-engine
docker pull FourLeafClover
docker run FourLeafClover
```
#### ubuntu
```
apt-get install  docker-engine
docker pull FourLeafClover
docker run FourLeafClover
```
